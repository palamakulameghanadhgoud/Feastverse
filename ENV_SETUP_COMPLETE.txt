QUICK START
═══════════════════════════════════════════════════════════════════

1. Create both .env files above
2. Setup MongoDB Atlas (get connection string)
3. Generate SECRET_KEY
4. Install dependencies:
   Backend:  cd Feastverse/backend && pip install -r requirements.txt
   Frontend: cd Feastverse/feastverse && npm install

5. Run servers:
   Backend:  python run.py (in backend folder)
   Frontend: npm run dev (in feastverse folder)

6. Open: http://localhost:5173
